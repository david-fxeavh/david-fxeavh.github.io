PhotoSwipe 图片查看器

一个功能强大、响应式的JavaScript图片库，支持触摸手势和优雅的过渡效果

如何使用 PhotoSwipe

  1. 引入 PhotoSwipe 的 CSS 和 JavaScript 文件
  2. 添加必需的 HTML 结构（PSWP 容器）
  3. 创建图片画廊的 HTML 结构
  4. 使用 JavaScript 初始化 PhotoSwipe
// 初始化 PhotoSwipe 的示例代码 const lightbox = new PhotoSwipeLightbox({ gallery: '#my-gallery', children: 'a', pswpModule: () => import('https://cdn.jsdelivr.net/npm/photoswipe@5.3.7/dist/photoswipe.esm.js') }); lightbox.init();
